# Dotfiles

my dotfiles.

# Getting Started

using `curl`

```
curl -sSL https://raw.githubusercontent.com/hidekuro/dotfiles/master/install.sh | bash
```

or wget

```
wget -qO - https://raw.githubusercontent.com/hidekuro/dotfiles/master/install.sh | bash
```
# Prerequisite

- wget / curl
- bash / zsh
- brew (in macOS)
- sudo
- git
